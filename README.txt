Project1 “A Simple Two-Dimensional Drawing System”

1.Make Project1
2.The program will automatically read the test.scene file.
3.Input the width and length for showing window so the program will clip for this window
4.Type in the method you want to use for drawing lines: DDA or Bresenham
5.Choose which polygon you want to transform or just transform All by typing in ‘9’
6.Chosse which transformation you want to apply:
(1)translation
(2)scaling
(3)rotation
(4)self_rotation
(5)self_scaling
7.The showing window will create the final results.
8.Press 's' to make a new transformation.
